How to use Bulk Import
  1. Download the sample CSV file template below
  2. Fill in your user data following the format in the sample file
  3. Make sure all required fields are filled: user_id, email, Full_name, password
  4. Upload your completed CSV file using the form below
CSV Format Requirements
  • user_id: Unique username (no spaces)
  • email: Valid email address (must be unique)
  • Full_name: User's full name
  • password: User's password (will be encrypted)
  • role: User role (optional, default: "User"). Valid values: User, Admin
  • status: Account status (optional, default: "active"). Valid values: active, inactive, suspended
  • BIO: User biography (optional)
Note: Users with duplicate usernames or emails will be skipped. Make sure your CSV has unique values for these fields.
